# Example protoype for Retrieval-Augmented Generation


## Prerequisites

- python3
- pip
- ollama

Models

- mistral:7b
- nomic-embed-text


## How to get it running

1. `pip install -r requirements.txt`
2. `ollama pull mistral`
3. `ollama pull nomic-embed-text`
4. `streamlit run Home.py`
